## installation
   -clone this repository
```bash
  run npm install
```
## start dev server
```bash
 run npm start
```
## enviroment variables
```bash
 include .env file
.env
    -DB_CONNECTION=<replace with url for mongodb>
```
 ## Deploy to Heroku
You can also deploy this app to Heroku:
Edit your `Procfile` and add the following line:
    web: node app.js
